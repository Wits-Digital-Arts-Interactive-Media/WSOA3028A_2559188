WEEK 6: Website Analysis

These are the websites I have been considering to analyse for my Essay. Each provides a service within South Africa.

PC Builder

https://pcbuilder.co.za

Pros:
  • Text: The text of the website is leagible and easy to read. Font colour changes when hovering around buttons to keep it readable.
  • Buttons: They have a hover feature on the buttons that lets the user know that their mouse is on the button. They also blend well with the rest of the page.
  • Interactive images: The product images highlight themselves around the border to indicate what the user is interested in.
  • Cookies: Does not request for cookies, user information and data remain confidential.
Cons:
  • Visual Graphics: The red, white, and blue colours do not blend so well. They are all bright and make it a bit difficult to read stuff.
  • Semantic Markup: From what I could tell from the inspector, the site makes use of many div containers. Though they help in making things flow, too much content can confuse the user.

Adoozy Power

https://www.adoozypower.co.za

Pros:
  • Load screens: The site has a loading screen which informs the user whether or not the site is working or not. This keeps the user invested.
  • Buttons: The buttons are simple and easy to navigate with. They highlight when the user hovers their mouse over them.
  • Cookies: This site does not have cookies, keeping the user's data private.
  • Aesthetic and Graphics: Neatly layed out visual elements that assist in keeping the user engaged. well taken photos of product.
  • Text: Legeable text that is easy to read and provides good descriptions of the products available.
Cons:
  • Usability and scalibility: Can be used on other devices like a cell phone, but loses it's proper layout making things squished on the screen.
  • Other than that there is nothing really bad about this site that I have scanned.

PROCARE

https://www.adoozypower.co.za

Pros:
  • Aesthetic: Cohesive and consistent aesthetic throughout the page. There is no contrasting colours and this helps in enforcing the "wellness" message of the page.
  • Text: The text is legeable and can be understood clearly
  • Cookies: The site asks permission for cookies and allows the user preference over what data they want to share with the site.
  • User communication: The information and layout of the site is layed out in such a way that it users are not over-whelmed.
  • Navigation: Everything is ordered and labelled correctly, making it easier for users to find what they need.
Cons:
  • Interactivity: There are not many interactive elements, it might be a challenge in keeping a user engaged.
  • Buttons: The buttons do not indicate to the user thata they can influence change on the site.
